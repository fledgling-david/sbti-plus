# SBTI人格测试项目

## 项目简介

SBTI人格测试是一种融合恶搞、自嘲与社交属性的轻量级测试产品，基于中文互联网流行的人格测试概念，为用户提供有趣的赛博人格分析。

### 核心特征
- 精简的题目数量（10题）
- 有限的选项设置（A/B/C三选一）
- 幽默化的测试结果（以互联网热词或抽象情绪标签为主）
- 复古Windows风格与霓虹赛博朋克相结合的视觉体验

## 技术栈

### 前端
- React 18+ + TypeScript
- Tailwind CSS
- Context API（状态管理）

### 后端
- Python 3.9+
- FastAPI
- Uvicorn

## 本地开发环境搭建

### 前端开发
1. 安装依赖
   ```bash
   npm install
   ```

2. 启动开发服务器
   ```bash
   npm run dev
   ```
   访问 http://localhost:5173

### 后端开发
1. 进入后端目录
   ```bash
   cd backend
   ```

2. 创建虚拟环境
   ```bash
   python3 -m venv venv
   ```

3. 激活虚拟环境
   ```bash
   # Windows
   venv\Scripts\activate
   
   # Linux/Mac
   source venv/bin/activate
   ```

4. 安装依赖
   ```bash
   pip install -r requirements.txt
   ```

5. 启动后端服务器
   ```bash
   uvicorn main:app --reload
   ```
   访问 http://localhost:8000

## 测试流程示例

### 测试流程1：卷王人格

**答题选项：**
1. A - 加班或学习新技能
2. A - 加倍努力，一定要完成目标
3. A - 用来展示自己的成就和生活
4. A - 据理力争，一定要说服对方
5. C - 觉得恋爱麻烦，不如搞钱
6. A - 学习或工作到深夜
7. A - 制定详细计划，努力实现目标
8. A - 加班是常态，为了更好的未来
9. A - 职场内卷相关
10. A - 立即开始下一个目标

**预期结果：**
- 人格类型：卷王
- 符号：👑
- 描述：极致内卷，永不满足
- 评语：你是职场/学习中的卷王，永远在追求更高的目标。别人休息时你在努力，别人努力时你在加倍努力。虽然可能会让周围的人感到压力，但你的成就也让人望尘莫及。不过，记得偶尔也要放松一下哦～

## 核心功能

1. **欢迎页**：包含产品介绍、视觉引导与开始测试按钮，添加复古电脑启动动画效果
2. **答题页**：实现题目轮播展示，配备带有复古进度条样式的答题进度指示器，选项按钮有悬停与点击反馈动画
3. **结果页**：展示人格测试结果、详细解析，实现结果海报生成功能，提供清晰的社交分享引导
4. **API接口**：提供题库数据的读取与返回接口，开发选项权重计算逻辑，根据用户答题结果进行人格类型匹配

## 注意事项

- 本项目仅供娱乐，测试结果不具有科学依据
- 部署时请确保后端服务正常运行，前端API地址配置正确
- 如需修改题库或人格类型，请同时更新前端和后端的数据
